# 📘 Lilavati Maths Classes

A modern and responsive educational website built using **HTML, CSS, JavaScript, and Vite** for showcasing mathematics coaching classes and academic services.

The project is designed with a clean UI, smooth animations, modern typography, and a professional landing-page experience for students and parents.

---

# ✨ Features

* 🎯 Professional educational landing page
* 📱 Fully responsive design
* 🎨 Modern UI with gradient backgrounds
* ⚡ Fast development using Vite
* 🧮 Interactive mathematics-themed design
* 🌐 Smooth navigation and animations
* 🔥 Optimized frontend structure
* 🎓 Coaching institute showcase layout

# 🚀 Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/DarkKnight-404/lelavatiMaths2.git
```

## 2️⃣ Navigate to Project Folder

```bash
cd lelavatiMaths2
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Run Development Server

```bash
npm run dev
```

The application will start on:

```bash
http://localhost:5173
```

---

# 📦 Build for Production

```bash
npm run build
```
